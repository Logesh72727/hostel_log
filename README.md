# Smart Hostel Entry System

A Flutter application for managing hostel entry/exit using biometric authentication and Firebase.

## Features
- Admin login via Firebase Auth
- Register students with fingerprint ID and photo
- Real-time view of entry/exit logs
- Search and filter logs
- Student profiles with activity history

## Tech Stack
- Flutter
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## Setup
See `docs/firebase_setup.md` for Firebase configuration.
Run `flutter pub get` then `flutter run`.
